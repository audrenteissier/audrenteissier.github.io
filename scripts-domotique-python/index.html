Multiroom Bose Python

Script python permettant de commander des enceintes Bose Soundtouch depuis Domoticz mais également de les associées en maître et esclave pour créer un vrai multiroom.

Ce script s’appuie sur une librairie très complète basée sur l’api officiel de bose. Elle permet:

– Multiroom
– Commande des presets (boutons sur l’enceinte)
– Commande de volume
– Mettre des musiques de sources différentes ( spotifty, url)
– Commandes : pause, play stop, passer une chansons…
– Mise en route des enceintes

https://easydomoticz.com/forum/viewtopic.php?t=4912
